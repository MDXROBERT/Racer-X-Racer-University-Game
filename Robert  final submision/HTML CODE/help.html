How to Play

The game is also made in such way that the first run everything is much faster and harder to play. But as you lose once it slows down and becomes easier to play. In order to Play the famous Racer X Racer game you will firstly need to learn the controls. In this case the controls are uparrow for moving up/forward, left arrow for moving left, right arrow for moving right and finally down arrow for moving down/backward. In this game there are there are multiple items that you can collect. The first item is the coins which boost your score by 200 points. The second item is a hourglass which slows down the time for 5 seconds. The third is a shield called divineintervention which protects you from the obstacles for 5 seconds. The fourth is a healthpack which gives 20 healh upon pickup.
